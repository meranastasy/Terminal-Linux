Первое ДЗ 
Linux terminal (GitBash) commands

=========

1) Посмотреть где я - pwd
2) Создать папку - mkdir papka script
3) Зайти в папку - cd papka
4) Создать 3 папки - mkdir papka1 papka2 papka3
5) Зайти в любоую папку - cd papka1
6) Создать 5 файлов (3 txt, 2 json) - touch text1.txt text2.txt text3.txt text1.json text2.json
7) Создать 3 папки - mkdir papka{1..3}
8. Вывести список содержимого папки - ls
	вывести расширенный список содержимого - ls -la
	вывести список содержимого по дате создания - ls -lt
	вывести список по дате обращения к файлу - ls -lu
9) + Открыть любой txt файл - vim text.txt
10) + написать туда что-нибудь, любой текст:
	войти в режим редактирования - i
	написать что-нибудь (abirvalg)
11) + сохранить и выйти. 
	выйти из режима редактирования - esc
	сохранить - :w
	сохранить под новым именем - :sav newtext.txt
	сохранить и выйти - :wq
12) Выйти из папки на уровень выше - cd ..
	выйти на два уровня выше - cd ../..
	переключиться на последнюю посещенную папку - cd -
13) 	переместить любые 2 файла, которые вы создали, в любую другую папку - mv text1.txt text2.txt papka1
14) скопировать любые 2 файла, которые вы создали, в любую другую папку - cp text3.txt text4.txt papka2
15) Найти файл по имени - find text.txt
16) просмотреть содержимое в реальном времени (команда grep) - tail -f text.txt | grep privet
	выйти ctrl+c
17) вывести несколько первых строк из текстового файла - head -3 text.txt
18) вывести несколько последних строк из текстового файла - tail -3 text.txt
19) просмотреть содержимое длинного файла (команда less) - less text.txt
	выйти - q
20) вывести дату и время - date
